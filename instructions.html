How to play — The Countdown

Quick instructions to start playing the puzzle game. Scroll for examples, tips and troubleshooting.

Quick start

  1. Click Generate Numbers to create the six numbers you will use to solve the target number.
  2. Click Get Target Number to generate the target number.
  3. Click Start Game to begin the timer and open the workingout areas.
  4. Use the on-screen keypad (or the keyboard) to build equations in the input box. Press OK (or Enter) to submit your equation
  5. The result of you equation will be displayed in the console where you will be able to use the number to help solve the target.

Controls

Generate Numbers Get Target Number Start Game Keypad (0–9, +, -, ×, ÷) OK / Enter / Back Undo

Rules

  • You may only use numbers that appear in the console (generated random numbers and the results from your equations. Each number can be used only once — then it becomes unavaiable (unless you create a new result which becomes available).
  • Allowed operators: addition +, subtraction -, multiplication *, and division /.
  • Parentheses and decimal points are disabled in the simplified keypad (but you can still type advanced expressions on desktop if enabled).
  • When you create a result, that result is added to the available numbers and can be used in later steps.
  • The goal is to reach the target number exactly before the timer runs out.

Example working

Available numbers: 2, 3, 3, 5, 50, 60 — Target: 58

      3 + 3 = 6     → add result 6 back to pool
      6 + 2 = 8     → add result 8 back to pool
      8 + 50 = 58   → target reached!

Input & console behaviour

  • The game uses one persistent input box. Submit a calculation, the result appears in the console and the input clears for your next calculation.
  • All results are kept in the console so you have a history of attempts.
  • If you enter a number that isn’t available, you’ll see an Error message which will appear in the console area. Dont worry any results will be displayed when you write another equation. message in the console.

Mobile & accessibility tips

  • The built-in keypad is optimised for mobile use so the native keyboard will not pop up while using the game keypad.
  • Buttons and keypad are large enough for touch; Landscape orientation for best layout.

Troubleshooting

  • No numbers appear — Press Generate Numbers again. If the problem repeats, reload the page.
  • Target looks too easy — Target is generated to be solvable with available numbers. If you want stricter targets, choose a different difficulty mode (if available).
  • Input seems unresponsive — Make sure the keypad was created (Start Game must be pressed). Check console for errors if you are debugging.

Scoring & winning

  • Win by producing the target number before the timer ends.
  • The winning message shows how many seconds remained when you solved it.
  • There is no penalty for extra steps — focus on getting to the exact target.

Reset / New game

  1. After a game ends (win or timeout), press Play again or refresh the page to restart.
Tip: if you want to practice without the timer, set a long difficulty time or pause the timer in the developer controls.

Still stuck? Contact the developer or check the in-game console for debug messages.
